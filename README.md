# Hermes REST API Example
- Hermes의 REST API 기반 동작 예시 코드입니다.
- Hermes의 Battery 상태를 확인하는 메서드 및 충전독으로 이동하여 충전하는 메서드가 있습니다.
- 상세한 내용은 `utils.py`를 확인하시면 좋습니다.

## Test Environment
- Python 3.6.9
- requests 2.27.1

## How to Use
- $ git clone https://github.com/WeGo-Robotics/Hermes_REST_API.git
- $ cd Hermes_REST_API
- $ python3 utils.py
